# Random Password Generator

Generate strong, secure passwords in your browser—fast and easy.  

## Features
- Generate **two passwords** at a time.  
- Custom length (4–32 characters).  
- Include uppercase, lowercase, numbers, symbols.  
- Works entirely **locally**, no server needed.  

## Usage
1. Open `index.html` in a browser.  
2. Set password options (length, character types).  
3. Click **Generate Password**.  
4. Copy your secure passwords.  

## File Structure
index.html # Main page
style.css # Styling
normalize.css # CSS reset
main.js # Password generation logic


## Notes
- JavaScript must be enabled.  
- All passwords are generated locally.
